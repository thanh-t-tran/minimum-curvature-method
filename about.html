Main Help About

The minimum curvature method is used to write VBA code in xlsm for survey calculation.

User defined functions starting with mc are intended to be used in excel sheets.

Here are the functions included in the Excel file

Notation:

  • MD: Measure Depth
  • I1: Incination 1 (degrees)
  • I2: Incination 2 (degrees)
  • A1: Aziimuth 1 (Degrees)
  • A2: Aziimuth 2 (Degrees)

Minimum curvature method Functions

Function Explain
mcVertial(I1, I2, A1, A2, DL) calculates the TVD
mcEast(I1, I2, A1, A2, DL) calculates the Easting
mcNorth(I1, I2, A1, A2, DL) calculates the Northing
DLS30Deg(I1, I2, DeltaAz, DL) calculates Dogleg for 30 length units
DLS100Deg(I1, I2, DeltaAz, DL) calculates Dogleg for 100 length units
DL = MD2-MD1 the measure depth between survey station 1 and 2 unit
