VBA Functionss

The minimum curvature method is used to write VBA code in xlsm for survey calculation.

User defined functions starting with prefix mc are intended to be used in excel sheets.

Here are the functions included in the Excel file

Minimum curvature method Functions

Function Explain
mcVertial(I1, I2, A1, A2, DL) Calculate the TVD
mcEast(I1, I2, A1, A2, DL) calculate the Easting
mcNorth(I1, I2, A1, A2, DL) Calculate the Northing
DLS30Deg(I1, I2, DeltaAz, DL) Calculate Dogleg for 30 length units
DLS100Deg(I1, I2, DeltaAz, DL) Calculate Dogleg for 100 length units

Where:

  • MD1: Measured Depth of survey station 1
  • MD2: Measured Depth of survey station 2
  • I1: Incination 1 (degrees) of survey station 1
  • I2: Incination 2 (degrees) of survey station 2
  • A1: Aziimuth 1 (degrees) of survey station 1
  • A2: Aziimuth 2 (degrees) of survey station 2
    • DL = MD2 - MD1 - Course length
    The length unit is determined by the input unit (meter or foot)

Reference:

1. Sawaryn, S.J. and Thorogood, J.L. 2005. A Compendium of Directional Calculations Based on the Minimum Curvature Method. SPE Drilling & Completion 20 (1): 24-36. SPE-84246-PA.https://doi.org/10.2118/84246-PA

2. https://www.drillingformulas.com/minimum-curvature-method/



Interpolation Functions

These functions will scan the first column of SurveyRange down until the empty row, recalculate the survey then calculate the interpolation of required parameters based on given TVD or MD in the middle point.

The SurveyRange requires a data range is a survey table with predefined format for calculation: The survey table contains at least 6 columns, which are MD, Inclination, Azimuth, TVD, North, East so that the interpolation functions can operate correctly. The table's final row must be left empty, serving as a marker for the functions that the array of chosen data has finished.

Although given only the top left cell it still scans down to get survery data from a wider range.

Functions take 2 parameters:

- First parameter (SurveyRange) is the top left cell of the survey table. If a range is given it still uses only the top left cell then expands to survey range.

- Second parameter (MD/TVD) is Measured Depth/True vertical depth which it needs to be mapped from.
