Main Help About

The minimum curvature method is used to write VBA code in xlsm for survey calculation.

User defined functions starting with mc are intended to be used in excel sheets.

Here are the functions included in the Excel file

Notation:

  • MD1: Measured Depth 1
  • MD2: Measured Depth 2
  • I1: Incination 1 (degrees)
  • I2: Incination 2 (degrees)
  • A1: Aziimuth 1 (Degrees)
  • A2: Aziimuth 2 (Degrees)
  • DL = MD2 - MD1

Minimum curvature method Functions

Function Explain
mcVertial(I1, I2, A1, A2, DL) Calculate the TVD
mcEast(I1, I2, A1, A2, DL) calculate the Easting
mcNorth(I1, I2, A1, A2, DL) Calculate the Northing
DLS30Deg(I1, I2, DeltaAz, DL) Calculate Dogleg for 30 length units
DLS100Deg(I1, I2, DeltaAz, DL) Calculate Dogleg for 100 length units

Interpolation Functions from MD

These functions will scan the first column of data range down until the empty row, recalculate the survey then calculate the interpolation of required parameters based on given tvd or td in the middle point.

The survey required a data range ís a survey table with predefined format for calculation.

Although given only the top left cell it still scans down to get data from a wider range.

Functions take 2 parameters

- First parameter is the top left cell of the survey table. If a range is given it still uses only top left cell then expands to survey range.

- Second parameter is measured depth/ tvd which needs to be mapped from.

Function Explain
mcGetInclinationfromMD(SurveyRange, MD) Calculate inclination from MD
mcGetAzimuthfromMD(SurveyRange, MD) Calculate Azimuth from MD
mcGetTVDfromMD(SurveyRange, MD) Calculate TVD from MD
mcGetEastfromMD(r, MD) Calculate Eastthing from MD
mcGetNorthfromMD(r, MD) Calculate Northhing from MD
mcGetDLS100fromMD(r, MD) Calculate DLS 100 from MD
mcGetDLS30fromMD(r, MD) Calculate DLS 30 from MD
