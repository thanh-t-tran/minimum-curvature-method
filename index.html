Minimum Curvature Method

By Tran Tuan Lam

Over View

This Excel file is about the calculation of the well survey using minimum curvature method and reverse the process when either the Measure Depth value or True Verticle Depth (TVD) is known.

VBA functions is stored in this work book. Macro should be enable for calculation.
These functions can be used as normal Excel function to carry out survey calculation, meaning the formula can be copy/paste amongst the cells.
Function name has prefix mc; for example function mcVertical calculate vertical distance between 2 survey stations.

Each of the cells use for calculation is color coded.
The Yellow cell is the your input, where as the others color is the output of its respective area (i.e. Local Coordinate, DLS)

How to use the file

The survey table contains at least 6 columns, which are MD, Inclination, Azimuth, TVD, North, East so that the functions can operate correctly.
The table's final row must be left empty, serving as a marker for the functions that the array of chosen data has finished.
All the cells outside the data range is free to use.
The unit is determined by the input unit.
I.E if the input of MD is in meter, the output will be in meter and vice versa.
